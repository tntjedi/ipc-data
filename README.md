# Visualization / Result Files

Use:

```text id="q8x7nv"
viz.py
```

to inspect any recorded run.

The script loads three CSV files for each run:

```text id="w4m1ft"
*_summary.csv
*_path.csv
*_motion.csv
```

---

## File Contents

### summary.csv

Contains final mission metrics such as:

* Time to goal
* Total distance travelled
* Goal threshold reached
* Max speed
* Max acceleration
* Max pitch rate
* Max yaw rate

these summary.csv files can be used to reffer the values mentioned above, they are not logs so its readable 

### path.csv

Time-history of vehicle position:

* time_sec
* x
* y
* z
* goal_distance

Used for trajectory and convergence plots.

### motion.csv

Time-history of vehicle motion:

* time_sec
* speed
* acceleration
* omega_y_pitch
* omega_z_yaw

Used for dynamic response plots.

---

## Plots Generated by viz.py

### 1. XY Trajectory

Top-view path of vehicle motion.

### 2. Linear Speed vs Time

Forward speed profile during mission.

### 3. Pitch & Yaw Angular Rates vs Time

Turning / orientation correction effort.

### 4. Acceleration vs Time

Motion smoothness and control aggressiveness.

### 5. Goal Distance vs Time

How quickly the vehicle converges to target.

### 6. Depth Profile (X-Z)

Forward motion versus depth change.
